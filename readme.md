# Requisitos
- Tener Node 10 o superior

# Instalacion

- Correr el comando para instalar todas las dependencias:
```
npm install
````

## Correr tests:
```
npm test
````
## Correr la api:
```
npm start
````

## Probar la api:

Se puede probar el endpoint desde la documentacion hecha en Swagger desde esta url:
```
http://localhost:3000/api-docs
```
